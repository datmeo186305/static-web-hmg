This is static website. It is built with [Webpack](https://webpack.js.org).

## Required

Node v12.18.0

## Install node_modules
```
npm install
```

## Run dev

```
npm start
```

## Build

```
npm run build:dist
```

## Run dist

```
npm run start:dist
```
